# Ansible Автоматизация

Проект содержит Ansible плейбуки и роли для автоматизации настройки серверов и развертывания приложений в Kubernetes окружении.


## О проекте

Этот проект предоставляет инфраструктуру как код для развертывания и управления:
- Kubernetes кластером
- Сервером мониторинга и CI/CD
- Стеком мониторинга (Prometheus, Grafana, Node Exporter)
- Инструментами CI/CD (GitLab Runner)

## Предварительные требования

- Ansible >= 2.9
- Python 3.6+
- SSH доступ к целевым серверам
- Terraform (для создания инфраструктуры)
- SSH ключи для доступа к серверам
